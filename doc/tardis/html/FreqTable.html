Module FreqTable

module FreqTable: sig .. end
Frequency table of symbols found in sources.
Author(s): Barthelemy Delemotte
type t = float array
The main type. A 256-table of probabilities. The indexes of the table corresponds to the value of the binary symbols (from 0 to 255).
val create : Source.t -> t
create src returns the frequency table corresponding the source src.
Raises Invalid_argument if src contains symbols not in [0..255].
